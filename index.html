Search Algorithms

Search Algorithms Overview

Linear Search: A simple search algorithm that checks each element in the list until the desired element is found or the list ends.

When to use: When the list is unsorted or small, as it does not require additional memory and is straightforward to implement.

Binary Search: A more efficient algorithm that requires the list to be sorted. It repeatedly divides the search interval in half.

When to use: When the list is sorted, as it significantly reduces the number of comparisons needed.

Ternary Search: Similar to binary search but divides the array into three parts instead of two, which can be more efficient in some scenarios.

When to use: When the list is sorted and you can afford the extra comparisons. It can outperform binary search in specific cases.
